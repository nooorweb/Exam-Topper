# Smart Prep MCQs

Expo React Native app for competitive exam preparation with MCQ practice,
vocabulary study, notes, past papers, local progress storage, and optional
Supabase sync.

## Prerequisites

- Node.js LTS is recommended. Expo SDK 51 works best on Node 18 or 20.
- npm
- Expo Go on your Android/iPhone for quick testing

## Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Optional: copy `.env.example` to `.env` and add Supabase values if you want
   login/cloud sync.

3. Start Expo:
   ```bash
   npm start
   ```

4. Scan the QR code with Expo Go, or press `a` in the terminal for Android.

## Useful Commands

```bash
npm start
npm run android
npm run web
npm run typecheck
```
